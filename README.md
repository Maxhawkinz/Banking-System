# 🏦 Banking System (Java Project)

## 📌 Overview
The **Banking System** is a Java-based application that simulates real-world banking operations.  
It allows users to create accounts, perform transactions, and manage financial data efficiently.

This project demonstrates core **Object-Oriented Programming (OOP)** concepts and basic banking functionalities like deposits, withdrawals, and passbook generation.

---

## 🎯 Aim
To design and implement a banking system that manages customer accounts and transactions efficiently while ensuring security and operational constraints.

---

## 🚀 Features
- 🆕 Account Creation with validation
- 💰 Deposit Money
- 💸 Withdraw Money with daily limit
- 📊 Balance Inquiry
- 📄 Account Information Display
- 📘 Passbook Generation (date-wise)
- 🔒 Secure Transaction Handling

## ⚙️ How It Works (Algorithm)

1. Initialize banking system with account storage  
2. Display menu for operations  
3. Account Creation:
   - Generate unique account number  
   - Store customer details  
4. Deposit:
   - Validate account  
   - Add amount  
   - Record transaction  
5. Withdrawal:
   - Check balance  
   - Enforce daily limit  
   - Deduct amount  
6. Passbook:
   - Filter transactions by date  
   - Display history
